## 購物指南

### 如何選擇
1. **正品保證** - 建議優先選擇品牌官方網站
2. **價格比較** - 不同平台可能有不同優惠
3. **配送服務** - 注意配送時間和運費
4. **售後服務** - 查看退換貨政策

### 支付方式
- 信用卡支付
- 超商取貨付款
- 第三方支付（Line Pay、街口支付等）
- ATM轉賬

### 配送時間
- 本島：通常1-3個工作日
- 外島：可能需要額外時間
- 國際配送：視品牌和平台而定

## 購物小貼士

1. **關注促銷活動** - 雙11、週年慶等大型促銷
2. **使用比價工具** - 比較不同平台價格
3. **查看用戶評價** - 了解商品真實反饋
4. **注意退換貨政策** - 保留購物憑證
